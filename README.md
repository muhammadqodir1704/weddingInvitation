# Wedding Invitation Project

## 🎵 Musiqa qo'shish yo'riqnomasi

### 1-usul: Avtomatik yuklash (Tavsiya etiladi)

1. **Musiqangizni `public` papkasiga ko'chiring**
2. **Fayl nomini `wedding-music.mp3` ga o'zgartiring** (yoki kodda fayl nomini o'zgartiring)
3. **Loyihani qayta ishga tushiring**

### 2-usul: Drag & Drop

1. **Loyihani oching:** http://localhost:3002
2. **Musiqa tugmasini bosing** (yuqorida Volume2 ikonasi)
3. **Musiqangizni sudrab tashlang** yoki **"Fayl tanlash"** tugmasini bosing
4. **Audio faylingizni tanlang**

### Qo'llab-quvvatlanadigan formatlar:
- MP3
- WAV
- AAC
- OGG

### Fayl hajmi:
- Maksimal: 50MB
- Tavsiya: 5-10MB

## 🚀 Loyihani ishga tushirish

```bash
# Loyihani ishga tushirish
npm run dev

# Build qilish
npm run build

# Production'da ishga tushirish
npm run start
```
